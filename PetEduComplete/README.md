# PetEdu - 电子宠物教学系统 🐾

一款用于课堂激励的桌面端电子宠物应用，学生通过答题、做作业、上课表现换取积分，用积分兑换宠物粮食实现宠物成长。

## 🎯 功能特性

### 学生端
- 🐾 **宠物养成** - 6 个成长阶段（蛋→幼年→童年→青少年→成年→老年）
- 💰 **积分系统** - 答题/作业/表现获得积分
- 🍖 **兑换系统** - 积分兑换粮食和道具
- 📊 **成长记录** - 查看宠物成长进度

### 教师端
- 👨‍🏫 **学生管理** - 班级学生列表
- 📊 **积分发放** - 按原因发放积分（答题/作业/表现）
- 🎁 **奖励发放** - 发放粮食和道具奖励
- 🏆 **积分排行** - 查看学生积分排名

## 🔧 环境要求

### 必需软件
- **Qt**: 5.15 或更高版本（Widgets 和 Sql 模块）
- **CMake**: 3.16 或更高版本
- **C++ 编译器**: MSVC 2019+ / GCC 9+ / Clang 10+

### 检查环境
```bash
# 检查 Qt 版本
qmake --version

# 检查 CMake 版本
cmake --version

# 检查编译器
g++ --version  # 或 cl
```

## 🎮 使用说明

### 学生操作流程
1. 启动程序，默认进入学生视图
2. 查看自己的宠物状态（积分/粮食/经验）
3. 点击"喂食宠物"消耗粮食增加经验
4. 点击"积分兑换"用积分兑换粮食
5. 宠物经验满后自动升级

### 教师操作流程
1. 点击底部"教师面板"按钮切换视图
2. 从下拉列表选择学生
3. 选择积分原因（答题/作业/表现）
4. 输入积分数量，点击"发放积分"
5. 或点击"发放奖励"直接给粮食/道具

## 📊 成长系统

### 等级划分
| 等级 | 阶段 | 外观 |
|------|------|------|
| 1-4 | 蛋 | 🥚 |
| 5-14 | 幼年期 | 🐣 |
| 15-29 | 童年期 | 🐤 |
| 30-49 | 青少年期 | 🐔 |
| 50-59 | 成年期 | 🦚 |
| 60 | 老年期 | 🦅 |

### 兑换规则
- **10 积分 = 1 粮食**
- **50 积分 = 6 粮食**（批量优惠）
- **1 粮食 = 5 经验**

### 升级经验
- 升级所需经验 = 当前等级 × 100
- 例如：1 级→2 级需要 100 经验，2 级→3 级需要 200 经验

## 🔧 高级配置

### 修改兑换比例
编辑 `src/main.cpp` 中的 `exchangePointsToFood` 函数

### 修改升级经验
编辑 `checkLevelUp` 函数中的经验计算公式

### 添加新学生
编辑 `TeacherPanel` 构造函数中的学生列表
