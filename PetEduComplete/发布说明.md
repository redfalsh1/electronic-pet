## 🎮 使用说明

### 学生操作流程

1. **启动程序** - 双击运行或命令行启动
2. **查看宠物** - 默认显示学生视图，可以看到宠物（🥚）
3. **喂食宠物** - 点击"🍖 喂食宠物"按钮
4. **兑换粮食** - 点击"💰 积分兑换"，选择兑换方案
5. **宠物升级** - 经验满 100 后自动升级，外观变化

### 教师操作流程

1. **切换视图** - 点击底部"👨‍🏫 教师面板"按钮
2. **选择学生** - 从下拉列表选择学生（张三/李四/王五/赵六）
3. **发放积分** - 选择原因（答题/作业/表现），输入积分，点击"发放积分"
4. **发放奖励** - 点击"🍖 粮食"或"✨ 道具"直接发放

## 📊 游戏数值

### 宠物成长阶段

| 等级 | 阶段 | 外观 |
|------|------|------|
| 1-4 | 蛋 | 🥚 |
| 5-14 | 幼年期 | 🐣 |
| 15-29 | 童年期 | 🐤 |
| 30-49 | 青少年期 | 🐔 |
| 50-59 | 成年期 | 🦚 |
| 60 | 完全体 | 🦅 |

### 兑换规则

- **10 积分 = 1 粮食**
- **50 积分 = 6 粮食**（批量优惠，省 4 积分）
- **1 粮食 = 5 经验**

### 升级经验

- 1 级→2 级：100 经验
- 2 级→3 级：200 经验
- n 级→n+1 级：n×100 经验

## 📞 技术支持

### 常见问题

**Q: 提示找不到 python 命令？**
A: 检查 Python 是否正确安装，或将 Python 添加到 PATH 环境变量

**Q: PyQt5 安装失败？**
A: 手动运行 `pip install PyQt5`，或使用国内镜像：
```batch
pip install PyQt5 -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**Q: 程序启动后看不到窗口？**
A: 检查任务栏，窗口可能最小化了

**Q: 数据丢失？**
A: 检查 `C:\Users\<用户名>\petedu_data.json` 是否存在
