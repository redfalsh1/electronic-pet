## 🔧 构建要求

### 必需软件
1. **Qt 5.15+** - 下载地址：https://www.qt.io/download
   - 安装时选择：Qt 5.15.x → MSVC 2019 64-bit
   - 勾选：Qt Widgets、Qt SQL 模块

2. **Visual Studio 2019/2022** - 下载地址：https://visualstudio.microsoft.com/
   - 安装时选择："使用 C++ 的桌面开发"

3. **CMake 3.16+** - 下载地址：https://cmake.org/download/

### 检查环境
打开命令提示符，运行：
```batch
qmake --version
cmake --version
cl
```

## 🚀 构建步骤

### 方法 1：使用 Qt Creator（推荐）

1. 打开 Qt Creator
2. 文件 → 打开文件或项目
3. 选择 `CMakeLists.txt`
4. 点击"配置项目"
5. 按 `Ctrl+B` 构建
6. 按 `Ctrl+R` 运行

### 方法 2：使用命令行

```batch
cd C:\Users\xqf\.openclaw\workspace-eda_tech_lead\PetEduComplete

REM 创建构建目录
mkdir build
cd build

REM 配置（使用 Qt 的 CMake 工具）
cmake .. -G "Visual Studio 16 2019" -A x64 ^
  -DCMAKE_PREFIX_PATH="C:\Qt\Qt5.15.2\5.15.2\msvc2019_64"

REM 编译
cmake --build . --config Release

REM 运行
.\Release\PetEdu.exe
```

### 方法 3：使用 Qt 命令行工具

打开 **Qt 5.15 MSVC 2019 64-bit Command Prompt**：

```batch
cd C:\Users\xqf\.openclaw\workspace-eda_tech_lead\PetEduComplete
```

### 功能演示

#### 学生端
1. 查看宠物状态（积分/粮食/经验）
2. 点击"喂食宠物" → 消耗 1 粮食 → 经验 +5
3. 点击"积分兑换" → 10 积分换 1 粮食
4. 经验满 100 → 自动升级 → 宠物外观变化

#### 教师端
1. 切换"教师面板"
2. 选择学生（张三/李四/王五/赵六）
3. 选择原因（课堂答题/完成作业/课堂表现）
4. 输入积分数量 → 点击"发放积分"
5. 学生获得积分后可兑换粮食

## 📊 游戏数值

### 成长阶段
| 等级 | 阶段 | Emoji |
|------|------|-------|
| 1-4 | 蛋 | 🥚 |
| 5-14 | 幼年期 | 🐣 |
| 15-29 | 童年期 | 🐤 |
| 30-49 | 青少年期 | 🐔 |
| 50-59 | 成年期 | 🦚 |
| 60 | 完全体 | 🦅 |

### 兑换规则
- 10 积分 = 1 粮食
- 50 积分 = 6 粮食（优惠 20%）
- 1 粮食 = 5 经验

### 升级经验
- 1 级→2 级：100 经验
- 2 级→3 级：200 经验
- ...
- n 级→n+1 级：n×100 经验

## 🎯 使用场景

### 课堂激励
1. 老师提问，学生回答正确 → 发放 5-10 积分
2. 完成课后作业 → 发放 20-30 积分
3. 课堂表现优秀 → 发放 5-15 积分
4. 学生用积分兑换粮食 → 喂养宠物 → 宠物成长

### 班级管理
- 可添加多个学生（需修改代码）
- 查看积分排行
- 定期清零重置
