# 图片资源说明

## 所需图片资源

请将以下图片文件放置在此目录：

### 宠物相关
- `pet_placeholder.png` - 宠物默认显示图片 (建议尺寸：300x300)
- `background.png` - 主界面背景图片 (建议尺寸：1920x1080)

### 图标
- `food_placeholder.png` - 粮食默认图标 (建议尺寸：64x64)
- `prop_placeholder.png` - 道具默认图标 (建议尺寸：64x64)
- `icon_score.png` - 积分图标 (建议尺寸：32x32)
- `icon_food.png` - 粮食图标 (建议尺寸：32x32)
- `icon_exp.png` - 经验图标 (建议尺寸：32x32)

## 图片格式要求

- 格式：PNG (支持透明通道)
- 颜色模式：RGBA
- 建议大小：见上方标注

## 临时方案

开发阶段可以使用纯色占位图，后续替换为精美 UI 资源。
